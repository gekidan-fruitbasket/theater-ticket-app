# 更新履歴

このプロジェクトの変更履歴をここに記録します。

## [0.1.0] - 2026-01-02

### 追加
- **主要機能**
    - Next.js 14, Supabase, Tailwind CSS による基本環境の構築。
    - LINEログイン連携（シームレスな認証）。
    - ビジュアル座席表（空席確認・予約機能）。
    - ユーザープロフィール登録フロー（必須項目：本名、劇団員名）。

- **管理者機能**
    - 管理者ダッシュボードの構築（`/admin`）。
    - リアルタイム予約リスト表示。
    - 受付名簿用CSVエクスポート機能。
    - 管理者による予約キャンセル・削除機能。

- **インフラ・ドキュメント**
    - Vercelへのデプロイ設定。
    - アプリ仕様書（README.md）の作成。
